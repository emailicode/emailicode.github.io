1. What is MySQL

MySQL is a relational database system used to store and manage structured data.

ConceptDescription
DatabaseContainer holding tables
TableStructure containing rows and columns
RowA single record
ColumnAn attribute of the record
Primary KeyUnique identifier for each row

6. Creating a Table

CREATE TABLE users (
    id INT,
    name VARCHAR(100),
    email VARCHAR(100)

7. Showing Tables

SHOW TABLES;

8. Viewing Table Structure

DESCRIBE users;

9. Primary Key

A Primary Key uniquely identifies each row in a table.

CREATE TABLE users (
    id INT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100)
);

10. Auto Increment

AUTO_INCREMENT automatically increases the value of the ID column.

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100)
);

11. Inserting Data

INSERT INTO users (name,email)
VALUES ('John','john@email.com');

12. Retrieving Data

SELECT * FROM users;

13. Filtering Data

SELECT * FROM users
WHERE id > 2;

14. Updating Data

UPDATE users
SET email='new@email.com'
WHERE id=1;

15. Deleting Data

DELETE FROM users
WHERE id=3;

16. Dropping a Table

DROP TABLE users;

17. Example Full Workflow

mysql -u root -p

CREATE DATABASE company_db;

USE company_db;

CREATE TABLE employees (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    department VARCHAR(50),
    salary INT
);

INSERT INTO employees (name,department,salary)
VALUES ('John','IT',60000);

SELECT * FROM employees;

19. Best Practices

  • Always use a primary key
  • Use AUTO_INCREMENT for IDs
  • Use meaningful table names
  • Always include WHERE when updating or deleting
  • Choose appropriate data types
